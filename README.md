# BinCombiner
a GUI program to create Flash Image for Burning
(posted by guitarbug @ 2015.6.17)

*support Flash Size : 1, 2, 4, 8, 16, 32MB;
*support Flash Partition Num : 1-6;
*support user define Partition Size and File Offset;
